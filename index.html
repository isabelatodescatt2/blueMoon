Instruções de Configuração
Resolvendo Problemas de Configuração
1 Problema de Porta

O React tenta usar a porta 3000, mas o Node.js já está usando essa porta.

Solução: Configure o React para usar a porta 3001 criando um arquivo .env na pasta do frontend com: 
PORT=3001
2 Warnings do ESLint

Variáveis importadas mas não utilizadas nos componentes.

Solução: Remova as importações não utilizadas ou adicione comentários para suprimir os warnings: 
// eslint-disable-next-line
3 Executando Backend e Frontend Juntos

Use o concurrently para executar ambos os serviços simultaneamente.

No package.json do backend, adicione: 
"scripts": {
  "dev": "nodemon server.js",
  "client": "npm start --prefix ../frontend",
  "dev:all": "concurrently \"npm run dev\" \"npm run client\""
}
4 Configurando o Proxy

Para que o frontend se comunique com o backend, configure o proxy.

No package.json do frontend, adicione: 
"proxy": "http://localhost:3000"
